# sockets-cpp

# Library with socket classes:
## TCP Client
## TCP Server
## UDP Multicast/Unicast

# Sample socket apps using these classes:
## TCP Client
```bash
$ ./clientApp -a <ipAddr> -p <port>
```
## TCP Server
```bash
$ ./serverApp -p <port>
```
## UDP Multicast
```bash
$ ./mcastApp -m <multicastAddr> -p <port>
```
## UDP Unicast
```bash
$ ./unicastApp -a <ipAddr> -l <localPort> -p <remotePort>
```
